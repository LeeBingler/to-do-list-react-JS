# to-do-list-react-JS

This application is a basic todo list in React-JS.


# Motivation
Make a reel project with JS, NPM and React

# Build/Run

#### Requirements

- Node.js
- NPM
- React

```javascript

/* First, Install the needed packages */
npm install

/* Then start the React app */
npm start

/* To run the tests */
npm run test

```
